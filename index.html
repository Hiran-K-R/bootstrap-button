Active & Disable Button

Active

If You create a Normal button Its a Active Button. how ever you can put .active class to specify it

Active Button < a href="#" class="btn btn-primary active" role="button" aria-pressed="true" > Active Button < /a >

Disable

Make buttons look inactive by adding the boolean attribute to any element.

< button type="button" class="btn btn-primary" disabled > Disabled Button < /button >
